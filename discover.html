Question-1: What are the different ways to select an element in the DOM?

There are multiple ways to do so. Like as getElementById(), getElementsByClassName(),getElementsByTagName(),querySelector(),querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets or sets HTML content inside an element, innerText gets or sets text content inside an element, textContent gets or sets text content hidden and visible both inside an element

Question-3: What is event delegation in the DOM?

Event delegations is technique whre parent node handles all the child nodes.For example if we make a new button then we will have to add new eventListener for that and we'll have to continue this process. But using event.target we can make it more efficient and we won't have to add new eventListener every time. Here this event.target works as event delegation

Question-4: What is event bubbling in the DOM?

Event bubbling is a method where an event starts from a target node/point and bubbles up to it's parent node until it reached to the core html element.To prevent it we use event.stopPropagation()

Question-5: How do you create, add, and remove elements using JavaScript?

For creating element we use document.createElement(), for removing we use removeChild() or remove() and for ading we use appendChild().
